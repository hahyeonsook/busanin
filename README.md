# BusanIn

부산의 여러 관광지와 맛집들을 소개하는 게시물을 올릴 수 있는 웹 사이트입니다. 사용자들은 댓글을 통해 서로 소통할 수 있습니다.

## Program Stacks

**Backend**

- Python 3.7
- Django 2.2.5
- Postgres
- AWS Elastic Beanstalk
- AWS S3

**Frontend**

- css
- HTML
- Tailwinds CSS
